# Flyme Auto 语音交互区 · 原型集

一组基于 Flyme Auto(魅族车机)设计语言的高保真网页原型。核心:**无边框、固定宽度(~392–424px)、高度随内容动态生长**的语音交互区 —— 出现内容时把右侧地图卡向右推开。**所有内容只在语音区内呈现。**

> 📖 设计语言、技术架构、版权约定等完整说明见 **`项目交接文档.md`**。

---

## 快速运行

```bash
cd "new avatar demo"
python3 -m http.server 8139
# 打开 http://localhost:8139/ 选任意 html
```
双击 html 也能开;用本地服务器图片/canvas 才正常。截图取景用 **1400×900** 视口。
每页右上角有调试面板,点按钮重放动效。

| `生成能力 × 场景地图.html` | 三种生成能力(组合 → 参数 → 结构)分别落到哪些场景:✅ 我们做过 / 🚗 车机常见 |

> 后两张口径一致(都是三档:组合→参数→结构,不含"像素生成");用 1600×900 视口取景。

## 三条必守规则

1. **只在 `new avatar demo` 文件夹改**;`桌面宠物` 是只读备份。
2. 新场景沿用同一套设计语言:无边框语音区 + 顶栏识别/回复 + 头像→Aicy 小球 + 地图推开 + 动态长高;地图卡**在即将出内容时**才推。
3. 用真实 Flyme/高德视觉、公共领域诗词;不抄版权歌词。
